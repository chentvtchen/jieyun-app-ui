## noDataTip 空状态
----
### 基础用法


```html
<no-data-tip :showFlag="true"></no-data-tip>
```
### 属性
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| showFlag  | 是否展示   | Boolean  |  —  | false  |
| msg       | 显示的消息 | String   |  —  | 暂无消息 |
| imgUrl    | 图片地址   | String   |  —  |  —  |
| top       | 距离顶部的像素大小   | Number   |  —  |  —  |
